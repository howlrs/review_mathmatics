## 高等学校数学（数学I, A重点）超速習 学習要項（全96時間）

### 1. 指導の基本方針と学習目標

#### 1.1. 学習目標
高校数学の基礎となる**数学Iの全範囲**（数と式、二次関数、図形と計量、データの分析）を網羅し、さらに論理的思考が求められる**数学Aの一部**（集合と命題、場合の数と確率の基礎）を習得します。

#### 1.2. プログラミングとの連携戦略（重点）
中学校数学に続き、高校数学では特に「集合と命題」「関数」「データ分析」の分野でプログラミングとの連携を深めます。

| 数学トピック         | プログラミング関連概念（Rust/Goなど）                                        |
| :------------------- | :--------------------------------------------------------------------------- |
| **集合と命題、論理** | ブール論理、型システム、形式検証、必要条件/十分条件（条件判定）              |
| **二次関数/グラフ**  | グラフ描画アルゴリズム、最適化問題（最大値/最小値探索）                      |
| **三角比**           | 幾何学的計算（2D/3Dエンジン）、角度表現（弧度法への準備）                    |
| **データの分析**     | 統計ライブラリの理解、データ分散の分析（分散、標準偏差）                     |
| **場合の数/確率**    | アルゴリズムの計算量分析（組合せ）、モンテカルロ法などの確率シミュレーション |

#### 1.3. 指導の構造
学習は会話履歴で確立したテンプレートに従い、各セッションは以下の構造で進めます。

1. **基礎情報と基礎問題10問**：定義、公式、概念の確認に集中し、「**困ったときは『定義』に還れ**」を徹底します。
2. **頻出応用問題10問とTips**：問題解決のアプローチ（「**複雑な問題は『素因数分解』せよ**」）を適用し、複数の解法や論理的根拠を考察します。

### 3. 学習を進める上でのポイント

高校数学は、中学校で学んだ基礎の上に、より抽象的で複雑な概念を導入します。特に数学Iは、その後の全ての科目の基盤となるため、96時間で完了させるには、知識の詰め込みではなく、**概念の構造的な理解**が不可欠です。

**抽象化と具体化のサイクル：**
例えば、「集合と命題」の学習は、プログラミングにおけるデータ型や型の検証、そして条件分岐の厳密な論理を理解するための強力なツールとなります。公式を学ぶ際も、「**なぜ？は理解のエンジン**」の格言に従い、定義や証明プロセス（例：余弦定理の導出）をしっかりと追うことで、応用力が飛躍的に向上します。
