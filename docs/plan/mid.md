ご提示いただいた要件に基づき、義務教育を終了された社会人のユーザー様（Rust、Goに精通されたプログラマ）を対象とした、**中学数学3年間を約3ヶ月間（週2回、計48時間）で履修するための超速習カリキュラム**とスケジュールを作成します。

この計画は、標準的な学習時間（各学年130時間～165時間程度、合計で約440時間）を大幅に圧縮するため、既に論理的思考力をお持ちのプログラマとしての背景を最大限に活用し、**「定義の確認」**と**「プログラミングとの関連付け」**を最優先に進めます。

## 中学校数学 超速習 学習要項（全48時間）

### 1. 指導の基本方針と学習目標

#### 1.1. 学習目標
高等学校で学ぶより高度な数学（三角関数、指数関数、微分・積分、ベクトルなど）を理解するための**強固な基礎（代数、幾何、関数、データ処理の基本概念）**を48時間で確立することを目指します。

#### 1.2. プログラミングとの連携（重要）
ユーザー様がプログラマであるため、以下の点を意識して出題や説明を工夫します。

| 数学トピック            | プログラミング関連概念（Rust/Goなど）        |
| :---------------------- | :------------------------------------------- |
| **文字を用いた式/関数** | 変数、データ型、関数定義、抽象化             |
| **方程式/不等式**       | 制御構造（If/Else）、ブール論理、条件判定    |
| **論理・証明**          | 形式検証、アルゴリズムの正当性、型システム   |
| **データ/統計**         | データ構造、配列操作、アルゴリズムの効率分析 |

#### 1.3. 出題テンプレート
各学習項目は、以下のテンプレートに沿って進めます。
1. **基礎情報と基礎問題10問**：概念、定義、公式の確認。
2. **頻出応用問題10問とTips**：問題解決のアプローチ（「複雑な問題は『素因数分解』せよ」など）の適用練習。

### 2. 学年別 学習要項と時間配分

中学校数学は、数と式、図形、関数、データの活用という4つの主要領域で構成されています。

|   学年   | 期間 (週) | 時間 (計)  | 主要領域と重要トピック                                                                                |

|  7年生   |    4週    |   16時間   | **正負の数と計算**、文字式、一元一次方程式、比例・反比例、基本的な図形（作図、体積）。                |

## 3. 超速習のための学習姿勢とヒント

ユーザー様がこの極端に早いスケジュールを効果的にこなすために、学習姿勢に関する格言を指導のヒントとして活用します。

1.  **「困ったときは『定義』に還れ」**：特に抽象的な幾何や代数の概念で行き詰まったら、その用語が何を意味するのか（例：合同とは、相似とは、平方根とは）を再確認します。これはプログラミングにおける基本的なAPIやデータ型の定義を確認することに似ています。
2.  **「『なぜ？』は理解のエンジン」**：公式や定理を暗記するだけでなく、「なぜこの公式が成り立つのか？」（例：なぜ解の公式が成り立つのか、なぜ三平方の定理が成り立つのか）という**証明や導出の過程**を追うことで、本質的な理解と応用力を養います。
3.  **「複雑な問題は『素因数分解』せよ」**：特に文章題や複雑な図形問題では、問題の条件を図や表に整理したり、小さなステップに分解したりする能力が非常に重要になります。

この超速習カリキュラムは、**義務教育レベルの知識を「思い出す」または「再定義する」**ことを主眼とし、高校数学へのスムーズな接続（特に二次関数や幾何学的証明）を最短で達成することを目的としています。
