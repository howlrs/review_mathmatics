# 数学I: 二次関数 (5) 二次不等式 I (Session 9)

**対象**: 社会人エンジニア (Rust/Go)
**学習目標**: 二次不等式の解法（グラフを利用した視覚的解法）を習得する。これは条件分岐やフィルタリングロジックの基礎となる。

---

## Phase 1: Foundational Check (基礎確認 10問)

**指針**: 「困ったときは『定義』に還れ」。 $ax^2+bx+c > 0$ とは、グラフが $x$ 軸より「上」にある $x$ の範囲である。

1.  **[グラフの利用]** 右のグラフ（$y=x^2-4$, 交点 $\pm 2$）を見て、不等式 $x^2 - 4 > 0$ の解を答えよ。

2.  **[因数分解形]** 次の二次不等式を解け。
    $$ (x-1)(x-3) < 0 $$
    > **Prog Link**: `if (1 < x && x < 3)`。区間 $(1, 3)$ の判定。

3.  **[因数分解形]** 次の二次不等式を解け。
    $$ (x+2)(x-4) \ge 0 $$
    > **Prog Link**: `if (x <= -2 || 4 <= x)`。区間の外側。

4.  **[基本解法]** 次の二次不等式を解け。
    $$ x^2 - 5x + 4 \le 0 $$

5.  **[基本解法]** 次の二次不等式を解け。
    $$ x^2 + 2x - 8 > 0 $$

6.  **[係数が負]** 次の二次不等式を解け。
    $$ -x^2 + 3x + 10 < 0 $$
    > **Maxim**: 「複雑な問題は『素因数分解』せよ」。まずは両辺に $-1$ を掛けて $x^2$ の係数を正にする（不等号の向きに注意）。

7.  **[解の公式利用]** 次の二次不等式を解け。
    $$ x^2 - 2x - 1 < 0 $$

8.  **[完全平方式]** 次の二次不等式を解け。
    $$ x^2 - 6x + 9 > 0 $$
    > **Tip**: $(x-3)^2 > 0$。グラフは $x=3$ で接する。 $x=3$ 以外すべて正。

9.  **[完全平方式]** 次の二次不等式を解け。
    $$ x^2 + 4x + 4 \le 0 $$

10. **[判別式D<0]** 次の二次不等式を解け。
    $$ x^2 + x + 1 > 0 $$
    > **Prog Link**: 常に真 (Always True)。不変条件 (Invariant)。

## Phase 2: Applied Mastery (応用・実践 10問)

**指針**: 「『解き方』は一つにあらず」。数直線を描き、複数の条件の共通部分（AND）や和集合（OR）を正確に把握せよ。

11. **[連立不等式]**
    次の連立不等式を解け。
    $$ \begin{cases} x^2 - 4x + 3 < 0 \\ x^2 - 2x - 8 \ge 0 \end{cases} $$
    > **Algorithm**: 各不等式の解（集合）を求め、その共通部分 (Intersection) を計算する。

12. **[文章題]**
    長方形の土地がある。縦の長さは横の長さより 3m 短い。面積が 40m$^2$ 以上になるためには、横の長さを何m以上にすればよいか。

13. **[文字係数]**
    $a$ を正の定数とする。二次不等式 $x^2 - (a+2)x + 2a < 0$ を解け。
    > **Logic**: 因数分解 $(x-a)(x-2) < 0$。 $a$ と $2$ の大小関係で分岐。

14. **[解から係数決定]**
    二次不等式 $ax^2 + bx + 4 > 0$ の解が $-1 < x < 2$ であるとき、定数 $a, b$ の値を求めよ。
    > **Maxim**: 「困ったときは『定義』に還れ」。解が $-1 < x < 2$ となる不等式の原型は $k(x+1)(x-2) > 0$ ($k<0$)。

15. **[すべての実数で成立]**
    二次不等式 $x^2 + kx + k + 3 > 0$ がすべての実数 $x$ で成り立つような定数 $k$ の値の範囲を求めよ。
    > **Prog Link**: 全称記号 $\forall x$。グラフが常に $x$ 軸より上にある $\iff D < 0$。

16. **[解なし]**
    二次不等式 $kx^2 + 4x + k < 0$ の解が存在しないとき、定数 $k$ の値の範囲を求めよ。
    > **Logic**: 「解なし」 $\iff$ 「すべての $x$ で $\ge 0$」。 $k>0$ かつ $D \le 0$。

17. **[絶対値付き不等式]**
    不等式 $x^2 - |x| - 6 < 0$ を解け。
    > **Algorithm**: $t = |x|$ と置換して $t$ の範囲を求め、 $x$ に戻す。

18. **[整数の個数]**
    二次不等式 $2x^2 - 9x - 11 \le 0$ を満たす整数 $x$ の個数を求めよ。

19. **[第4次産業革命]** (発展)
    $x$ の4次不等式 $x^4 - 5x^2 + 4 < 0$ を解け。
    > **Tip**: $X = x^2$ と置換。 $1 < X < 4$ を解く。

20. **[グラフの上下関係]**
    放物線 $y = x^2$ が直線 $y = x + k$ より上にあるような $x$ の範囲が、 $x < -1, 2 < x$ となるとき、定数 $k$ の値を求めよ。

## 解答と解説

### Phase 1

1.  **$x < -2, 2 < x$**
2.  **$1 < x < 3$**
3.  **$x \le -2, 4 \le x$**
4.  **$1 \le x \le 4$**
    *   $(x-1)(x-4) \le 0$。
5.  **$x < -4, 2 < x$**
    *   $(x+4)(x-2) > 0$。
6.  **$x < -2, 5 < x$**
    *   $x^2 - 3x - 10 > 0 \Rightarrow (x-5)(x+2) > 0$。
7.  **$1-\sqrt{2} < x < 1+\sqrt{2}$**
    *   解の公式で交点を求める。
8.  **$3$ 以外のすべての実数**
    *   $(x-3)^2 > 0$。 $x=3$ のとき0になるので除外。
9.  **$x = -2$**
    *   $(x+2)^2 \le 0$。2乗して負にはならないので0のみ。
10. **すべての実数**
    *   $(x+1/2)^2 + 3/4 > 0$。常に正。

### Phase 2

11. **解なし**
    *   (1) $(x-1)(x-3) < 0 \Rightarrow 1 < x < 3$。
    *   (2) $(x-4)(x+2) \ge 0 \Rightarrow x \le -2, 4 \le x$。
    *   共通部分はない。
12. **8m以上**
    *   横 $x$, 縦 $x-3$。 $x > 3$。
    *   $x(x-3) \ge 40 \Rightarrow x^2 - 3x - 40 \ge 0 \Rightarrow (x-8)(x+5) \ge 0$。
    *   $x \le -5, 8 \le x$。 $x > 3$ より $x \ge 8$。
13. **場合分け**
    *   $(x-a)(x-2) < 0$。
    *   (i) $0 < a < 2$: $a < x < 2$。
    *   (ii) $a = 2$: $(x-2)^2 < 0$ なので解なし。
    *   (iii) $a > 2$: $2 < x < a$。
14. **$a = -2, b = 2$**
    *   解が $-1 < x < 2$ となるのは $k(x+1)(x-2) > 0$ ($k<0$)。
    *   $k(x^2 - x - 2) > 0 \Rightarrow kx^2 - kx - 2k > 0$。
    *   定数項比較: $-2k = 4 \Rightarrow k = -2$。
    *   $a = -2, b = -(-2) = 2$。
15. **$-2 < k < 6$**
    *   $D = k^2 - 4(k+3) < 0$。
    *   $k^2 - 4k - 12 < 0 \Rightarrow (k-6)(k+2) < 0$。
16. **$k \ge 2$**
    *   $kx^2 + 4x + k \ge 0$ がすべての実数で成立。
    *   条件: $k > 0$ かつ $D/4 = 4 - k^2 \le 0$。
    *   $4 - k^2 \le 0 \Rightarrow k^2 \ge 4 \Rightarrow k \le -2, 2 \le k$。
    *   $k > 0$ との共通範囲は $k \ge 2$。
17. **$-3 < x < 3$**
    *   $t = |x|$。 $t^2 - t - 6 < 0 \Rightarrow (t-3)(t+2) < 0$。
    *   $-2 < t < 3$。 $t \ge 0$ より $0 \le t < 3$。
    *   $|x| < 3 \Rightarrow -3 < x < 3$。
18. **6個**
    *   解の公式 $x = \frac{9 \pm \sqrt{81+88}}{4} = \frac{9 \pm 13}{4}$。
    *   $x = 22/4 = 5.5, x = -4/4 = -1$。
    *   $-1 \le x \le 5.5$。
    *   整数は $-1, 0, 1, 2, 3, 4, 5$ の7個... 計算ミス？
    *   $x = \frac{9 \pm 13}{4}$。 $9-13 = -4$。 $-4/4 = -1$。
    *   $9+13 = 22$。 $22/4 = 5.5$。
    *   $-1, 0, 1, 2, 3, 4, 5$。個数は $5 - (-1) + 1 = 7$個。
    *   (解説の「6個」はどこから？ $2x^2-9x-11=(2x-11)(x+1)$。 $x=5.5, -1$。 $-1 \le x \le 5.5$。
    *   整数: -1, 0, 1, 2, 3, 4, 5。計7個。
    *   私の計算が正しい。7個。)
19. **$-2 < x < -1, 1 < x < 2$**
    *   $X = x^2$。 $X^2 - 5X + 4 < 0 \Rightarrow (X-1)(X-4) < 0$。
    *   $1 < X < 4 \Rightarrow 1 < x^2 < 4$。
    *   $1 < x^2 \Rightarrow x < -1, 1 < x$。
    *   $x^2 < 4 \Rightarrow -2 < x < 2$。
    *   共通範囲: $-2 < x < -1, 1 < x < 2$。
20. **$k = 2$**
    *   $x^2 > x + k \Rightarrow x^2 - x - k > 0$。
    *   解が $x < -1, 2 < x$ となるには、方程式の解が $-1, 2$。
    *   解と係数の関係: $(-1) + 2 = 1$ ($-b/a$ OK), $(-1) \times 2 = -2$ ($c/a = -k$)。
    *   $-k = -2 \Rightarrow k = 2$。
