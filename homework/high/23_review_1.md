# 数学I: 総合演習 I (Session 23)

**対象**: 社会人エンジニア (Rust/Go)
**学習目標**: Phase 1の前半（数と式、論理、二次関数）の重要概念を総復習し、定着度を確認する。

---

## Phase 1: Foundational Check (基礎確認 10問)

**指針**: 「困ったときは『定義』に還れ」。基本公式や定義を即座に引き出せるかチェックせよ。

1.  **[実数]** $\sqrt{2}$ が無理数であることを、背理法を用いて証明する際の仮定を述べよ。
    > **Logic**: 「有理数であると仮定する」。つまり $p/q$ （既約分数）と置く。

2.  **[集合]** $A = \{1, 2, 3, 4\}, B = \{3, 4, 5, 6\}$ のとき、 $A \cap B$ と $A \cup B$ を求めよ。

3.  **[命題]** 命題「$x > 0 \Rightarrow x^2 > 0$」の逆、裏、対偶を述べ、それぞれの真偽を判定せよ。
    > **Maxim**: 「対偶は元の命題と真偽が一致する」。

4.  **[不等式]** 不等式 $|2x - 1| < 3$ を解け。

5.  **[二次関数]** $y = 2x^2 - 8x + 5$ を平方完成し、頂点の座標を求めよ。

6.  **[最大・最小]** $y = -x^2 + 4x + 1$ ($0 \le x \le 3$) の最大値と最小値を求めよ。

7.  **[二次方程式]** $2x^2 - 3x - 1 = 0$ を解け。

8.  **[判別式]** 二次方程式 $x^2 + kx + 9 = 0$ が重解を持つような定数 $k$ の値を求めよ。

9.  **[二次不等式]** $x^2 - x - 6 > 0$ を解け。

10. **[絶対値]** $\sqrt{(x-1)^2}$ を絶対値記号を用いて表せ。また、 $x < 1$ のときどうなるか。
    > **Prog Link**: `abs(x-1)`。

## Phase 2: Applied Mastery (応用・実践 10問)

**指針**: 「複雑な問題は『素因数分解』せよ」。複数の単元（例：二次関数と論理）が融合した問題を解きほぐせ。

11. **[集合と論理]**
    実数全体を全体集合とし、条件 $p: |x| < 2$、条件 $q: x^2 - 2x - 3 < 0$ とする。命題 $p \Rightarrow q$ の真偽を集合の包含関係を用いて判定せよ。
    > **Logic**: $P \subset Q$ ならば真。

12. **[必要十分条件]**
    $x, y$ は実数とする。 $x^2 + y^2 = 0$ であることは、 $x=0$ かつ $y=0$ であるための何条件か。

13. **[二次関数の決定]**
    頂点が $(1, 2)$ で、点 $(2, 4)$ を通る二次関数の式を求めよ。

14. **[軸が動く最大・最小]**
    $y = x^2 - 2ax + 1$ ($0 \le x \le 2$) の最小値を $m(a)$ とする。 $m(a)$ を $a$ の場合分けで表せ。

15. **[解の配置]**
    二次方程式 $x^2 - 2ax + a + 2 = 0$ が、異なる2つの正の解を持つような定数 $a$ の値の範囲を求めよ。

16. **[絶対値付き方程式]**
    方程式 $x^2 - |x| - 2 = 0$ を解け。

17. **[文章題（最大化）]**
    直角を挟む2辺の長さの和が 20 である直角三角形の面積の最大値を求めよ。

18. **[不等式の証明]**
    $a > 0, b > 0$ のとき、 $(a+b)(1/a + 4/b) \ge 9$ を証明せよ。
    > **Tip**: 展開して相加相乗平均。

19. **[整数の性質]** (Math A)
    $n$ を整数とするとき、 $n^2$ を 4 で割った余りは 0 または 1 であることを証明せよ。

20. **[プログラミング]**
    二次方程式 $ax^2 + bx + c = 0$ の解を求める関数 `solve_quadratic(a, b, c)` を実装する際、 $a=0$ の場合や $D<0$ の場合をどのように処理すべきか、エラーハンドリングの観点から述べよ。
    > **Prog Link**: `Result<T, E>` や `error` 返り値の設計。

## 解答と解説

### Phase 1

1.  **$\sqrt{2}$ は有理数である**
2.  **$A \cap B = \{3, 4\}, A \cup B = \{1, 2, 3, 4, 5, 6\}$**
3.  **逆: $x^2 > 0 \Rightarrow x > 0$ (偽, $x=-1$), 裏: $x \le 0 \Rightarrow x^2 \le 0$ (偽), 対偶: $x^2 \le 0 \Rightarrow x \le 0$ (真)**
    *   元の命題は真。
4.  **$-1 < x < 2$**
    *   $-3 < 2x - 1 < 3 \Rightarrow -2 < 2x < 4$。
5.  **$2(x-2)^2 - 3$, 頂点 $(2, -3)$**
6.  **最大値 5 ($x=2$), 最小値 1 ($x=0$)**
    *   $y = -(x-2)^2 + 5$。
    *   $x=0 \to 1, x=3 \to 4$。
7.  **$x = \frac{3 \pm \sqrt{17}}{4}$**
8.  **$k = \pm 6$**
9.  **$x < -2, 3 < x$**
    *   $(x-3)(x+2) > 0$。
10. **$|x-1|$, $-(x-1) = 1-x$**

### Phase 2

11. **真**
    *   $P = \{x \mid -2 < x < 2\}$。
    *   $Q = \{x \mid -1 < x < 3\}$。
    *   $P \not\subset Q$ ($x=-1.5$ は $P$ に入るが $Q$ に入らない)。
    *   **訂正**: 偽。反例 $x=-1.5$。
12. **必要十分条件**
13. **$y = 2(x-1)^2 + 2$**
    *   $y = a(x-1)^2 + 2$。 $4 = a(1)^2 + 2 \Rightarrow a=2$。
14. **(i) $a<0: a^2+1$, (ii) $0 \le a \le 2: -a^2+1$, (iii) $a>2: a^2-4a+5$** ... 訂正
    *   (i) $a<0$: 軸左。 $x=0$ で最小 $1$。
    *   (ii) $0 \le a \le 2$: 軸内。 $x=a$ で最小 $-a^2+1$。
    *   (iii) $a>2$: 軸右。 $x=2$ で最小 $4-4a+1 = 5-4a$。
    *   (計算ミス多発注意。 $f(0)=1, f(2)=5-4a$)。
15. **$a > 2$**
    *   $D>0, \text{axis}>0, f(0)>0$。
16. **$x = \pm 2$**
    *   $t^2 - t - 2 = 0 \Rightarrow (t-2)(t+1)=0$。 $t=|x| \ge 0 \Rightarrow t=2$。
17. **50**
    *   $x, 20-x$。 $S = x(20-x)/2$。最大値 50 ($x=10$)。
18. **証明**
    *   $1 + 4a/b + b/a + 4 = 5 + 4a/b + b/a \ge 5 + 2\sqrt{4} = 9$。
19. **証明**
    *   $n=2k \to 4k^2 \equiv 0$。
    *   $n=2k+1 \to 4k^2+4k+1 \equiv 1$。
20. **解説**
    *   $a=0$: 一次方程式として解くか、エラー（`InvalidArgument`）を返す。
    *   $D<0$: 実数解なし（`None` / `nil`）を返すか、複素数型を返す。
