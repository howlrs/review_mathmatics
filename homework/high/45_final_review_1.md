## Phase 1: Foundational Check (概念の再構築 10問)

**指針**: 単なる計算ではなく、「なぜそうなるか」という構造理解を問う。

1.  **[必要十分条件]** $x^2 = 1$ は $x = 1$ であるための何条件か。
    > **Answer**: 必要条件（十分条件ではない）。 $x=-1$ の可能性があるため。

2.  **[背理法]** $\sqrt{2}$ が無理数であることを証明する際、最初に何を仮定するか。
    > **Assumption**: $\sqrt{2}$ が有理数である（既約分数 $p/q$ で表せる）と仮定する。

3.  **[2次関数の決定]** 頂点が $(1, 2)$ で、点 $(0, 3)$ を通る2次関数を求めよ。
    > **Formula**: $y = a(x-1)^2 + 2$。

4.  **[判別式]** 2次方程式 $ax^2 + bx + c = 0$ が実数解を持たない条件を $D$ で表せ。
    > **Condition**: $D < 0$。

5.  **[剰余の定理]** $P(x)$ を $2x - 1$ で割った余りは $P(?)$ に等しいか。
    > **Value**: $1/2$。

6.  **[因数分解]** $x^3 + y^3 + z^3 - 3xyz$ の因数分解公式を記述せよ。
    > **Formula**: $(x+y+z)(x^2+y^2+z^2-xy-yz-zx)$。

7.  **[相加相乗平均]** $x > 0$ のとき、 $x + 1/x$ の最小値はいくつか。またその時の $x$ は。
    > **Value**: 2 ($x=1$)。

8.  **[絶対値]** $|x-1| < 3$ を不等式で解け。
    > **Solution**: $-3 < x-1 < 3 \Rightarrow -2 < x < 4$。

9.  **[解と係数の関係]** 3次方程式 $x^3 + ax^2 + bx + c = 0$ の3解を $\alpha, \beta, \gamma$ とするとき、 $\alpha + \beta + \gamma$ は何に等しいか。
    > **Value**: $-a$。

10. **[プログラミング]** 浮動小数点数における「桁落ち」とはどのような現象か。
    > **Tech**: 値が近い2つの数の引き算で、有効数字が急激に減る現象。

## Phase 2: Applied Mastery (構造的理解の深化 10問)

**指針**: 複数の単元をまたぐ問題や、抽象度の高い問題に取り組む。

11. **[論理と集合]**
    実数全体の集合 $R$ において、条件 $p: |x| < 1, q: x^2 - 2ax + 1 > 0$ を考える。 $p$ が $q$ であるための十分条件となるような定数 $a$ の範囲を求めよ。
    > **Logic**: $-1 < x < 1$ ならば常に $x^2 - 2ax + 1 > 0$。
    > $f(x) = x^2 - 2ax + 1$ の区間 $(-1, 1)$ での最小値が正であればよい。

12. **[2次関数の最大最小]**
    $x \ge 0, y \ge 0, 2x + y = 4$ のとき、 $x^2 + y^2$ の最大値と最小値を求めよ。
    > **Algorithm**: $y = 4-2x$ を代入して1変数化。定義域 $0 \le x \le 2$ に注意。

13. **[高次方程式と複素数]**
    方程式 $x^3 = 1$ の虚数解の一つを $\omega$ とするとき、 $\omega^{100} + \omega^{50} + 1$ の値を求めよ。
    > **Calculation**: $\omega^3=1$ より $\omega^{100}=\omega, \omega^{50}=\omega^2$。 $\omega^2+\omega+1=0$。

14. **[不等式の証明]**
    $a > 0, b > 0, c > 0$ のとき、 $(a+b)(b+c)(c+a) \ge 8abc$ を証明せよ。
    > **Proof**: 相加相乗平均を3回使う。 $(a+b) \ge 2\sqrt{ab}$ など。

15. **[整数と論理]**
    $n$ が整数のとき、 $n^2$ を3で割った余りは 0 または 1 であることを証明せよ。
    > **Proof**: $n = 3k, 3k+1, 3k+2$ で場合分け。

16. **[多項式の除算]**
    整式 $x^n - 1$ が $(x-1)^2$ で割り切れるための $n$ の条件（$n \ge 2$）はあるか。
    > **Logic**: $P(1)=0$ かつ $P'(1)=0$。 $P'(x) = nx^{n-1}$。 $P'(1)=n \ne 0$。割り切れない。

17. **[対称式]**
    $x + y = 4, xy = 2$ のとき、 $x^3 + y^3$ の値を求めよ。
    > **Calculation**: $(x+y)^3 - 3xy(x+y)$。

18. **[軌跡と領域]**
    放物線 $y = x^2 + kx + k^2$ の頂点 $P$ が動く領域を図示せよ（$k$ は実数全体）。
    > **Algorithm**: $x = -k/2, y = k^2 - k^2/2 - k^2/4$ などを計算して $k$ 消去。

19. **[2変数の不等式]**
    すべての実数 $x$ に対して $x^2 + ax + b > 0$ が成り立つ条件を $a, b$ で表せ。
    > **Condition**: 判別式 $D = a^2 - 4b < 0$ かつ（下に凸なので係数正）。つまり $b > a^2/4$。

20. **[プログラミング: 型システム]**
    Rust/Go のような静的型付け言語において、 `int` と `float` の暗黙の型変換を許さない設計思想の数学的・工学的メリットは何か。
    > **Tech**: 精度損失や意図しない振る舞い（等価判定など）をコンパイル時に防ぎ、堅牢性を高めるため。

11. **$-1 \le a \le 1$**
    *   軸 $x=a$。
    *   (i) $a < -1$: 最小値 $f(-1) = 1+2a+1 = 2a+2 > 0 \to a > -1$。矛盾。
    *   (ii) $-1 \le a \le 1$: 最小値 $f(a) = -a^2+1 > 0 \to a^2 < 1$。OK。
    *   (iii) $a > 1$: 最小値 $f(1) = 1-2a+1 = 2-2a > 0 \to a < 1$。矛盾。
    *   答え: $-1 < a < 1$。
    *   (端点チェック: $x$ の範囲は開区間 $(-1, 1)$。
    *   $a=1$ のとき $f(x) = (x-1)^2$。 $(-1, 1)$ で正か？ $x \ne 1$ なので正。OK。
    *   $a=-1$ のとき $f(x) = (x+1)^2$。 $(-1, 1)$ で正か？ $x \ne -1$ なので正。OK。
    *   答え: $-1 \le a \le 1$)
12. **最大 16/5, 最小 4** ... 訂正
    *   $y = 4-2x$。 $0 \le 4-2x \Rightarrow x \le 2$。 $0 \le x \le 2$。
    *   $f(x) = x^2 + (4-2x)^2 = x^2 + 16 - 16x + 4x^2 = 5x^2 - 16x + 16$。
    *   軸 $x = 16/10 = 8/5 = 1.6$。
    *   最小値: $x=1.6$。 $5(1.6)^2 - 16(1.6) + 16 = 12.8 - 25.6 + 16 = 3.2 = 16/5$。
    *   最大値: 軸から遠い端点 $x=0$。 $f(0)=16$。 ($x=2 \to 20-32+16=4$)。
    *   答え: 最大 16 ($x=0, y=4$), 最小 16/5 ($x=8/5, y=4/5$)。
