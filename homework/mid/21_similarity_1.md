## Phase 1: Foundational Check (基礎の確認) - 10問

**「スケーリング (Scaling)」**: 拡大・縮小しても変わらない性質（角度）と、変わる性質（長さ・面積）を区別します。

### Q1. 相似の定義
2つの図形が相似であるとは、どのような関係ですか。

<details>
<summary>解答と解説</summary>

**解答:** 一方の図形を拡大または縮小すると、他方の図形と合同になる関係。

**解説:**
形は同じで大きさが違う。
対応する角の大きさは等しく、対応する辺の長さの比は一定。

> **Programming Link:**
> *   **Vector Image:** SVGなどのベクター画像は、解像度を変えても（スケーリングしても）劣化しません。
</details>

### Q2. 相似比
相似な2つの図形の対応する辺の長さの比を何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 相似比 (Ratio of Similitude)

**解説:**
`1:2` ならば、すべての長さが2倍になっている。

### Q3. 三角形の相似条件 (1)
3組の辺の比がすべて等しいとき、2つの三角形は相似ですか。

<details>
<summary>解答と解説</summary>

**解答:** 相似である（二辺比夾角相等）

**解説:**

### Q5. 三角形の相似条件 (3)
2組の角がそれぞれ等しいとき、2つの三角形は相似ですか。

<details>
<summary>解答と解説</summary>

**解答:** 相似である（二角相等）

**解説:**
AA Similarity.
残りの1つの角も自動的に等しくなるため、3組の角が等しいことと同じ。最もよく使われる条件。
</details>

### Q6. 相似の中心
2つの相似な図形の対応する点を結ぶ直線がすべて1点Oで交わるとき、点Oを何といいますか。

<details>

### Q7. 平行線と線分の比 (1)

## Phase 2: Applied Mastery (応用と実践) - 10問

**「フラクタル」**: 部分と全体が自己相似になっている構造を見抜きます。

### Q11. 相似比の計算
相似な2つの三角形があり、対応する辺の長さがそれぞれ `4cm` と `6cm` です。小さい方の三角形の周の長さが `12cm` のとき、大きい方の周の長さは？

<details>
<summary>解答と解説</summary>

**解答:** `18cm`

**Tips:**
相似比 `4:6 = 2:3`。
周の長さの比も `2:3`。
`12 : x = 2 : 3` → `2x = 36` → `x = 18`。
</details>

### Q12. 縮尺の利用
縮尺 `1/25000` の地図上で、長さが `4cm` の道は、実際には何kmですか。

<details>
<summary>解答と解説</summary>

**解答:** `1km`

**Tips:**
`4cm × 25000 = 100000cm`。
`100000cm = 1000m = 1km`。

> **Programming Link:**
> *   **Coordinate System:** ワールド座標系とスクリーン座標系の変換です。
</details>

### Q13. 影の長さ（測量）
身長1.6mの人の影の長さが2mのとき、影の長さが10mの木の高さは何mですか。

<details>
<summary>解答と解説</summary>

**解答:** 8m

**Tips:**
太陽光線は平行とみなせるので、直角三角形が相似になる。
`1.6 : 2 = x : 10`
`2x = 16`
`x = 8`。
</details>

### Q14. 平行線と比（台形）
台形ABCD（AD // BC）の対角線の交点をOとします。AD=4, BC=8のとき、DO:BO を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `1 : 2`

**Tips:**
`△OAD ∽ △OCB`（砂時計型）。
相似比は `AD : CB = 4 : 8 = 1 : 2`。
よって `DO : BO = 1 : 2`。
</details>

### Q15. 平行線と比（クロス）
AB // CD // EF のとき、xの値を求めなさい。
AB=10, CD=15, EF=x。
（図省略：ABとCDが平行で、その間に交差する線があり、交点から平行線を引いてEFとするパターン）
**修正:** よくある「AB // EF // CD」のパターンとします。
AB=10, CD=15, ABとCDは平行。その間の線分上にEFがあり平行。
公式: `1/x = 1/10 + 1/15`。

<details>
<summary>解答と解説</summary>

**解答:** `6`

**Tips:**
「和分の積」の公式 `x = (10*15)/(10+15) = 150/25 = 6`。
証明は相似を2回使う。
`△AEB ∽ △CED` → `BE:ED = 10:15 = 2:3`。
`△BCD` において `EF // DC` なので `EF:DC = BE:BD = 2:5`。
`x : 15 = 2 : 5` → `5x = 30` → `x = 6`。

> **Programming Link:**
> *   **Resistor:** 並列回路の合成抵抗の式 `1/R = 1/R1 + 1/R2` と同じ形です。調和平均に関連します。
</details>

### Q18. 折り返しと相似
長方形ABCDを頂点Aが辺BC上の点Eに来るように折り返したとき、`△EBF ∽ △GCE` （FはAB上の折り目、GはCD上の点）となることを証明しなさい。

<details>
<summary>解答と解説</summary>

**解答:**
`∠B = ∠C = 90`度。
`∠FEG = 90`度（角Aが移動したもの）。
一直線上の角 `∠BEG = 180`度より、`∠BEF + 90 + ∠CEG = 180`。
よって `∠BEF + ∠CEG = 90`。
また `△EBF` の内角和より `∠BEF + ∠BFE = 90`。
ゆえに `∠CEG = ∠BFE`。
2角が等しいので相似。

**Tips:**
「直角がある場合の角度計算」による証明パターン。
</details>

### Q19. 相似の利用（円）
円の弦AB, CDが点Pで交わるとき、`△PAC ∽ △PDB` であることを証明しなさい。

<details>
<summary>解答と解説</summary>

**解答:**
円周角の定理（円周角は等しい）より `∠CAP = ∠BDP`。
対頂角より `∠APC = ∠DPB`。
2角相等で相似。
（方べきの定理の証明）
</details>

### Q20. 再帰的な図形
正三角形の中に、各辺の中点を結んで小さな正三角形を作ります。これを繰り返してできる図形の性質は？

<details>
<summary>解答と解説</summary>

**解答:** シェルピンスキーのギャスケット（フラクタル図形）

**Tips:**
無限に繰り返すと面積は0になるが、周囲の長さは無限大になる。
</details>
