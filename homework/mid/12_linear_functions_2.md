## Phase 1: Foundational Check (基礎の確認) - 10問

**「線形モデルの適用」**: 現実世界の現象を `y = ax + b` で近似（モデル化）し、問題を解決します。

### Q1. 直線の式の決定（復習）
傾きが `-2` で、点 `(1, 4)` を通る直線の式を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `y = -2x + 6`

**解説:**
`4 = -2(1) + b` → `b = 6`。
</details>

### Q2. 交点の意味
2つの直線の交点は、連立方程式の（　　）と一致する。

<details>
<summary>解答と解説</summary>

**解答:** 解

**解説:**
グラフ上の交点は、両方の式を同時に満たす点。
</details>

### Q3. 平行線の性質
2つの直線が平行であるとき、それらの（　　）は等しい。

<details>
<summary>解答と解説</summary>

**解答:** 傾き（変化の割合）

**解説:**
`y = ax + b` と `y = ax + c` は平行。
</details>

### Q4. 軸に平行な直線
点 `(3, 5)` を通り、x軸に平行な直線の式は？

<details>
<summary>解答と解説</summary>

**解答:** `y = 5`

**解説:**
y座標が常に一定。傾きは0。
</details>

### Q5. 軸に垂直な直線
点 `(3, 5)` を通り、x軸に垂直な直線の式は？

<details>
<summary>解答と解説</summary>

**解答:** `x = 3`

**解説:**
x座標が常に一定。傾きは定義されない（無限大）。
これは関数ではない（1つのxに無数のyが対応するため）。

> **Programming Link:**
> *   **Vertical Line:** 垂直線判定。傾き `m` で管理しているとゼロ除算が発生するため、別途フラグ管理するか、ベクトル形式 `ax + by + c = 0` で扱う必要があります。
</details>

### Q6. 変域と最大・最小
`y = 2x + 1` (`1 ≦ x ≦ 3`) の最大値を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** 7

**解説:**
右上がりなので、xが最大のときyも最大。
`2(3) + 1 = 7`。
</details>

### Q7. 切片の移動
直線 `y = 3x` をy軸方向に `-2` だけ平行移動させた直線の式は？

<details>
<summary>解答と解説</summary>

**解答:** `y = 3x - 2`

**解説:**
切片が `0` から `-2` に変わる。
</details>

### Q8. 変化の割合の符号
xが増加するとyが減少するのは、変化の割合が（　　）のときである。

<details>
<summary>解答と解説</summary>

**解答:** 負（マイナス）

**解説:**
右下がりのグラフ。
</details>

### Q9. 一次関数の判別
次の中から一次関数を選びなさい。
(a) `y = 2/x`, (b) `y = x/2`, (c) `y = x^2`

<details>
<summary>解答と解説</summary>

**解答:** (b)

**解説:**
(b) は `y = 1/2 x` なので一次関数。
(a) は反比例、(c) は二次関数。
</details>

### Q10. グラフの概形
`a > 0, b < 0` のとき、`y = ax + b` のグラフは第何象限を通らないか。

<details>
<summary>解答と解説</summary>

**解答:** 第2象限

**解説:**
右上がりで、y軸と負の部分で交わる。
右下（第4）、左下（第3）、右上（第1）を通る。
</details>

## Phase 2: Applied Mastery (応用と実践) - 10問

**「ダイヤグラム」**: 時刻と位置の関係を表すグラフ（Time-Distance Graph）は、スケジューリングや並行処理の可視化に役立ちます。

### Q11. ダイヤグラムの読み取り
縦軸に道のり、横軸に時間をとったグラフにおいて、直線の「傾き」は何を表していますか。

<details>
<summary>解答と解説</summary>

**解答:** 速さ

**Tips:**
道のり / 時間 = 速さ。
傾きが急なほど速い。
</details>

### Q12. 追いつく問題（グラフ）
弟が家を出て歩いていき、10分後に兄が自転車で追いかけました。
グラフ上で、兄の直線と弟の直線の交点は何を表していますか。

<details>
<summary>解答と解説</summary>

**解答:** 兄が弟に追いついた時刻と、家からの距離。

**Tips:**
位置と時間が一致する点。
</details>

### Q13. 往復のグラフ
A地点からB地点まで行き、すぐに戻ってきました。
行きは時速4km、帰りは時速6kmでした。
グラフの形はどうなりますか（山の形）。

<details>
<summary>解答と解説</summary>

**解答:** 行きの傾きより、帰りの傾き（絶対値）の方が急な山型になる。

**Tips:**
速いほうが傾きが急。帰りは距離が減るので傾きは負。
</details>

### Q14. 料金プランの選択
1ヶ月の使用量を `x`、料金を `y` とします。
プランA: 基本料1000円、1単位あたり20円
プランB: 基本料2000円、1単位あたり10円
`x` がいくつのとき料金が並びますか。

<details>
<summary>解答と解説</summary>

**解答:** 100単位

**Tips:**
`20x + 1000 = 10x + 2000`
`10x = 1000`
`x = 100`
</details>

### Q15. 動点と面積（変化の様子）
長方形ABCD（AB=4, BC=6）の周上を点PがA→B→C→Dと動くとき、三角形APDの面積 `y` の最大値は？

<details>
<summary>解答と解説</summary>

**解答:** 12

**Tips:**
底辺AD=6、高さAB=4 のとき最大。
`1/2 × 6 × 4 = 12`。
Pが辺BC上にある間はずっと最大値12を維持する。
</details>

### Q16. ロウソクの燃焼
長さ20cmのロウソクに火をつけると、1分間に0.5cmずつ短くなります。
`x` 分後の長さ `y` cmを表す式と、燃え尽きるまでの時間を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `y = -0.5x + 20`、40分後

**Tips:**
`0 = -0.5x + 20`
`0.5x = 20`
`x = 40`

> **Programming Link:**
> *   **Time to Live (TTL):** リソースが枯渇するまでの時間予測です。
</details>

### Q17. 2直線の交点と面積
`y = x + 2` と `y = -2x + 8` とx軸で囲まれた三角形の面積を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** 12

**Tips:**
交点: `x + 2 = -2x + 8` → `3x = 6` → `x = 2`。`y = 4`。高さ4。
x軸との交点: `(-2, 0)` と `(4, 0)`。底辺 `4 - (-2) = 6`。
面積: `6 × 4 ÷ 2 = 12`。
</details>

### Q18. 1次関数の利用（気温）
地上から100m上がるごとに気温が0.6℃下がるとします。
地上の気温が20℃のとき、地上 `x` mの気温 `y` ℃を表す式は？

<details>
<summary>解答と解説</summary>

**解答:** `y = -0.006x + 20`

**Tips:**
1mあたり `0.6 / 100 = 0.006` ℃下がる。
傾き `-0.006`。
</details>

### Q19. 区分関数のグラフ
「0〜3分は毎分200m、3〜5分は休憩、5〜10分は毎分100m」
この動きをグラフに表すと、どのような線になりますか。

<details>
<summary>解答と解説</summary>

**解答:** 急な右上がり → 水平 → 緩やかな右上がり の折れ線。

**Tips:**
速度の変化が傾きの変化として現れる。
</details>

### Q20. データの傾向線（回帰直線の直感）
散布図に直線を引いて傾向を見ると、`x` が増えると `y` も増える傾向がありました。この直線の傾きは正ですか、負ですか。

<details>
<summary>解答と解説</summary>

**解答:** 正

**Tips:**
正の相関がある場合、近似直線の傾きは正になる。

> **Programming Link:**
> *   **Linear Regression:** 最小二乗法などで求めた回帰直線 `y = ax + b` は、データの傾向を最もよく表す一次関数モデルです。
</details>
