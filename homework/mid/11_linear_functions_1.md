## Phase 1: Foundational Check (基礎の確認) - 10問

**「線形性 (Linearity)」**: 入力と出力が直線的な関係にあること。予測しやすく、制御しやすいシステムの理想形です。

### Q1. 一次関数の定義
`y` が `x` の一次関数であるとき、一般的にどのような式で表されますか。

<details>
<summary>解答と解説</summary>

**解答:** `y = ax + b` （`a, b`は定数、`a ≠ 0`）

**解説:**
`b = 0` のときは比例（`y = ax`）になる。つまり比例は一次関数の特別な場合。

> **Programming Link:**
> *   **Affine Transform:** アフィン変換（線形変換 + 平行移動）の1次元版です。
</details>

### Q2. 変化の割合
一次関数 `y = ax + b` において、`x` の増加量に対する `y` の増加量の割合（`yの増加量 / xの増加量`）は常に一定です。この値を何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 変化の割合（または傾き `a`）

**解説:**
どの区間をとっても一定。これが線形性の特徴。

> **Programming Link:**
> *   **Gradient:** 勾配。機械学習（勾配降下法）における重要な概念です。
</details>

### Q3. 傾きと切片
一次関数 `y = -3x + 5` のグラフの傾きと切片を答えなさい。

<details>
<summary>解答と解説</summary>

**解答:** 傾き: `-3`, 切片: `5`

**解説:**
*   **傾き (Slope):** `a` の値。右下がり。
*   **切片 (Intercept):** y軸との交点のy座標（`x=0`のときのyの値）。
</details>

### Q4. 変化の割合の計算
関数 `y = 2x - 3` について、`x` の値が `1` から `4` まで増加するときの変化の割合を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `2`

**解説:**
計算しなくても `a` の値そのもの。
確認:
`x=1` → `y=-1`
`x=4` → `y=5`
`y`の増加量: `5 - (-1) = 6`
`x`の増加量: `4 - 1 = 3`
変化の割合: `6 / 3 = 2`
</details>

### Q5. グラフの作成
`y = 2x - 1` のグラフを描くとき、最初にプロットすべき点はどこですか（最も簡単な点）。

<details>
<summary>解答と解説</summary>

**解答:** 切片 `(0, -1)`

**解説:**
まずy軸上の切片をとり、そこから「右に1、上に2」進んだ点をとって結ぶ。
</details>

### Q6. 平行なグラフ
`y = 3x + 2` のグラフと平行で、原点を通る直線の式は？

<details>
<summary>解答と解説</summary>

**解答:** `y = 3x`

**解説:**
平行 ⇔ 傾きが等しい。
</details>

### Q7. 式の決定 (1)
傾きが `4` で、点 `(0, -2)` を通る直線の式を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `y = 4x - 2`

**解説:**
点 `(0, -2)` は切片そのもの。
</details>

### Q8. 式の決定 (2)
変化の割合が `-2` で、`x = 3` のとき `y = 1` となる一次関数の式を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `y = -2x + 7`

**解説:**
`y = -2x + b` とおく。
`1 = -2(3) + b`
`1 = -6 + b`
`b = 7`
</details>

### Q9. 変域
関数 `y = -x + 3` について、`x` の変域が `-2 ≦ x ≦ 4` のときの `y` の変域を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `-1 ≦ y ≦ 5`

**解説:**
`x = -2` → `y = 5`
`x = 4` → `y = -1`
傾きが負なので、大小関係が逆転することに注意。
</details>

### Q10. x軸との交点
`y = 2x - 8` のグラフとx軸との交点の座標を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `(4, 0)`

**解説:**
x軸上の点は `y = 0`。
`0 = 2x - 8`
`2x = 8` → `x = 4`

> **Programming Link:**
> *   **Root Finding:** 方程式 `f(x) = 0` の解（根）を求めることと同じです。
</details>

## Phase 2: Applied Mastery (応用と実践) - 10問

**「補間と予測」**: 2点のデータから直線を引くことで、その間の値（補間）や未来の値（外挿）を予測できます。

### Q11. 2点を通る直線の式
2点 `(1, 3)`, `(3, 7)` を通る直線の式を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `y = 2x + 1`

**Tips:**
1. 傾きを求める: `(7-3) / (3-1) = 4/2 = 2`。
2. `y = 2x + b` に `(1, 3)` を代入: `3 = 2 + b` → `b = 1`。

> **Programming Link:**
> *   **Linear Interpolation (Lerp):** 2点間を直線で補間する `lerp` 関数の基礎です。
</details>

### Q12. 連立方程式との関係
連立方程式 `2x + y = 5`, `x - y = 1` の解は、グラフ上では何を表していますか。

<details>
<summary>解答と解説</summary>

**解答:** 2つの直線 `y = -2x + 5` と `y = x - 1` の交点の座標。

**Tips:**
幾何学的な意味（交点）と代数的な意味（連立方程式の解）が一致します。
</details>

### Q13. 直線の交点
2直線 `y = 2x - 3` と `y = -x + 6` の交点の座標を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `(3, 3)`

**Tips:**
`2x - 3 = -x + 6`
`3x = 9` → `x = 3`
`y = 2(3) - 3 = 3`
</details>

### Q14. 三角形の面積（座標）
直線 `y = -2x + 8` とx軸、y軸で囲まれた三角形の面積を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** 16

**Tips:**
切片（y軸との交点）: `(0, 8)` → 高さ8
x軸との交点: `0 = -2x + 8` → `x = 4` → 底辺4
面積: `4 × 8 ÷ 2 = 16`
</details>

### Q15. バネの伸び
あるバネにおもりをつるしたとき、おもりの重さ `x` g とバネの長さ `y` cm の関係が一次関数になるとします。
10gで12cm、20gで14cmのとき、何もつるさないときの長さ（自然長）は何cmですか。

<details>
<summary>解答と解説</summary>

**解答:** 10cm

**Tips:**
10g増えると2cm伸びる（傾き `0.2`）。
10g減らせば2cm縮む。
`12 - 2 = 10`。
式: `y = 0.2x + 10`。切片が自然長。

> **Programming Link:**
> *   **Bias:** 機械学習のニューロンモデル `y = wx + b` におけるバイアス `b` に相当します。入力が0のときの出力値です。
</details>

### Q16. 水そうの問題（排水）
満水の水そう（100L）から、毎分5Lずつ水を抜きます。`x` 分後の残りの水量 `y` Lを式で表し、変域も答えなさい。

<details>
<summary>解答と解説</summary>

**解答:** `y = -5x + 100` (`0 ≦ x ≦ 20`)

**Tips:**
傾きが負（減少）。
`y = 0` になるのは `x = 20` のとき。
</details>

### Q17. 動点の問題（グラフ化）
長方形ABCDの周上を点PがA→B→C→Dと動くとき、AからPまでの距離 `x` と、三角形APDの面積 `y` の関係を表すグラフの概形はどうなりますか。

<details>
<summary>解答と解説</summary>

**解答:**
1. A→B: 底辺APが増えるので、右上がりの直線。
2. B→C: 底辺AB（固定）、高さAD（固定）なので、水平な直線（一定）。
3. C→D: 底辺PDが減るので、右下がりの直線。
台形のようなグラフになる。

**Tips:**
区間ごとに挙動が変わる関数（区分線形関数）。
</details>

### Q18. 料金プランの比較（再）
基本料金0円で単価が高いプランAと、基本料金が高くて単価が安いプランBのグラフの交点は何を意味しますか。

<details>
<summary>解答と解説</summary>

**解答:** どちらのプランでも料金が同じになる使用量（損益分岐点）。

**Tips:**
交点より右側では、傾きの緩やかなプランBの方が安くなる。
</details>

### Q19. 3点が一直線上にある条件
3点 `(1, 2)`, `(3, 6)`, `(a, 10)` が一直線上にあるとき、`a` の値を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `a = 5`

**Tips:**
最初の2点の傾き: `(6-2)/(3-1) = 2`。
`(3, 6)` と `(a, 10)` の傾きも2になるはず。
`(10-6)/(a-3) = 2`
`4 = 2(a-3)`
`2 = a-3`
`a = 5`
</details>

### Q20. 垂直な直線の傾き（発展）
直線 `y = 2x` に垂直な直線の傾きはいくつですか。
（高校数学の範囲ですが、プログラマなら知っておくと便利）

<details>
<summary>解答と解説</summary>

**解答:** `-1/2`

**Tips:**
垂直な2直線の傾きの積は `-1` になる。
`a × a' = -1`。
ベクトル `(1, 2)` と `(2, -1)` の内積が0になることから導けます。
</details>
