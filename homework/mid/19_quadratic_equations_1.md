## Phase 1: Foundational Check (基礎の確認) - 10問

**「解の探索」**: 方程式を解くことは、条件を満たす変数の値を探索することです。二次方程式には通常2つの解があります。

### Q1. 二次方程式の定義
`ax^2 + bx + c = 0` (`a ≠ 0`) の形の方程式を何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 二次方程式

**解説:**
`a = 0` なら一次方程式になってしまう。
</details>

### Q2. 解の個数
二次方程式の解は、一般的にいくつありますか。

<details>
<summary>解答と解説</summary>

**解答:** 2つ

**解説:**
重解（1つに見えるが重複している）や、実数解なし（高校範囲の虚数解）の場合もあるが、基本は2つ。
次数と解の個数は一致する（代数学の基本定理）。
</details>

### Q3. 平方根の利用 (1)
`x^2 = 9` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = ±3`

**解説:**
`x = 3, -3`。
</details>

### Q4. 平方根の利用 (2)
`x^2 - 5 = 0` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = ±√5`

**解説:**
移項して `x^2 = 5`。
</details>

### Q5. 平方根の利用 (3)
`(x - 2)^2 = 9` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 5, -1`

**解説:**
`x - 2 = ±3`
`x = 2 ± 3`
`2 + 3 = 5`, `2 - 3 = -1`。
</details>

### Q6. 因数分解の利用 (1) 原理
`AB = 0` ならば、AとBについて何がいえますか。

<details>
<summary>解答と解説</summary>

**解答:** `A = 0` または `B = 0`

**解説:**
これが因数分解による解法の原理（Zero Product Property）。
どちらか一方が0なら積は0になる。

> **Programming Link:**
> *   **Short-circuit Evaluation:** `if (A == 0 || B == 0)` の論理です。
</details>

### Q7. 因数分解の利用 (2)
`(x - 2)(x + 3) = 0` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 2, -3`

**解説:**
`x - 2 = 0` または `x + 3 = 0`。
</details>

### Q8. 因数分解の利用 (3)
`x^2 + 5x + 6 = 0` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = -2, -3`

**解説:**
左辺を因数分解して `(x + 2)(x + 3) = 0`。
</details>

### Q9. 因数分解の利用 (4)
`x^2 - 4x = 0` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 0, 4`

**解説:**
共通因数 `x` でくくる。
`x(x - 4) = 0`。
`x` で割って `x = 4` だけにするのは間違い（`x=0`の可能性があるため割れない）。
</details>

### Q10. 重解
`x^2 - 6x + 9 = 0` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 3`

**解説:**
`(x - 3)^2 = 0`。
解は `3` だけ（重解）。
</details>

## Phase 2: Applied Mastery (応用と実践) - 10問

**「問題の帰着」**: 複雑な方程式も、式変形によって基本的な形（`X^2 = k` や `(x-a)(x-b)=0`）に帰着させます。

### Q11. 係数がある場合
`2x^2 - 12x + 18 = 0` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 3`

**Tips:**
まず両辺を2で割る（正規化）。
`x^2 - 6x + 9 = 0`。
</details>

### Q12. 移項が必要な場合
`x^2 + 3x = 10` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 2, -5`

**Tips:**
右辺を0にする。
`x^2 + 3x - 10 = 0`
`(x - 2)(x + 5) = 0`。
</details>

### Q13. 展開が必要な場合
`(x + 1)(x - 2) = 4` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 3, -2`

**Tips:**
`x = -1, 2` ではない！（右辺が0でないため）。
展開して整理する。
`x^2 - x - 2 = 4`
`x^2 - x - 6 = 0`
`(x - 3)(x + 2) = 0`。
</details>

### Q14. 平方根の利用（複雑）
`3(x + 1)^2 - 12 = 0` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 1, -3`

**Tips:**
`3(x + 1)^2 = 12`
`(x + 1)^2 = 4`
`x + 1 = ±2`
`x = -1 ± 2`。
</details>

### Q15. 因数分解（和と差の積）
`x^2 - 25 = 0` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = ±5`

**Tips:**
`(x + 5)(x - 5) = 0`。
または `x^2 = 25` としてもよい。
</details>

### Q16. 解からの逆算
二次方程式 `x^2 + ax + b = 0` の解が `2, 3` であるとき、`a, b` の値を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `a = -5, b = 6`

**Tips:**
解が `2, 3` ということは、式は `(x - 2)(x - 3) = 0` と因数分解できるはず。
展開すると `x^2 - 5x + 6 = 0`。
係数比較する。
（解と係数の関係）
</details>

### Q17. 連続する整数
連続する2つの正の整数の積が72のとき、この2つの整数を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** 8, 9

**Tips:**
`n(n + 1) = 72`
`n^2 + n - 72 = 0`
`(n + 9)(n - 8) = 0`
`n = -9, 8`。
正の整数なので `n = 8`。
よって `8, 9`。
</details>

### Q18. 動点と面積（二次方程式）
正方形ABCD（一辺10cm）の辺上を、点PはAからBへ、点QはBからCへ動きます。Pの速さは毎秒1cm、Qの速さは毎秒2cmです。
△PBQの面積が24cm^2になるのは何秒後ですか。（`0 < t < 5`）

<details>
<summary>解答と解説</summary>

**解答:** 4秒後

**Tips:**
t秒後: AP=`t`, PB=`10-t`, BQ=`2t`。
面積 `1/2 × (10-t) × 2t = 24`
`t(10-t) = 24`
`10t - t^2 = 24`
`t^2 - 10t + 24 = 0`
`(t - 4)(t - 6) = 0`
`t = 4, 6`。
変域 `0 < t < 5` より `t = 4`。
</details>

### Q19. 容積の問題
長方形の厚紙（縦10cm、横20cm）の四隅から正方形を切り取り、蓋のない箱を作ったら、底面積が75cm^2になりました。切り取った正方形の一辺の長さは？

<details>
<summary>解答と解説</summary>

**解答:** 2.5cm

**Tips:**
切り取る長さを `x` とする。
底面の縦 `10 - 2x`、横 `20 - 2x`。
`(10 - 2x)(20 - 2x) = 75`
`4(5 - x)(10 - x) = 75` ... 整数にならない？
`200 - 60x + 4x^2 = 75`
`4x^2 - 60x + 125 = 0`
解の公式が必要かも（次回）。
因数分解できるか確認:
`(2x - 5)(2x - 25) = 0`
`x = 2.5, 12.5`。
`10 - 2x > 0` より `x < 5`。
よって `x = 2.5`。
</details>

### Q20. 誤り訂正
「`x^2 = 3x` の両辺を `x` で割って `x = 3`」
この解法の誤りを指摘しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 0` の場合に0で割ることになるため。

**Tips:**
`x = 0` も解の一つだが、割ることで情報が失われる。
正しくは `x^2 - 3x = 0` → `x(x - 3) = 0`。

> **Programming Link:**
> *   **Edge Case:** ゼロ除算や境界値（0）の処理漏れは典型的なバグです。
</details>
