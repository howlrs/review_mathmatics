## Phase 1: Foundational Check (基礎の確認) - 10問

**「制約の交点」**: 複数の条件（方程式）を同時に満たす状態（解）を見つけるプロセスです。

### Q1. 二元一次方程式
次の方程式のうち、二元一次方程式であるものを選びなさい。
(a) `x + y = 5`, (b) `x^2 + y = 4`, (c) `2x - 3y + 1 = 0`, (d) `xy = 6`

<details>
<summary>解答と解説</summary>

**解答:** (a), (c)

**解説:**
変数が2つで、次数が1の方程式。
(b)は二次式、(d)は `xy` が2次なので不可。

> **Programming Link:**
> *   **Linear Constraint:** 線形計画法（Linear Programming）などで扱われる線形制約条件です。
</details>

### Q2. 連立方程式の解
`x + y = 7`, `2x - y = 2` の解として正しいものを選びなさい。
(a) `(2, 5)`, (b) `(3, 4)`, (c) `(4, 3)`

<details>
<summary>解答と解説</summary>

**解答:** (b) `(3, 4)`

**解説:**
両方の式に代入して成り立つか確認。
`3 + 4 = 7` (OK)
`2(3) - 4 = 6 - 4 = 2` (OK)
</details>

### Q3. 加減法 (1)
次の連立方程式を解きなさい。
`x + y = 8`
`x - y = 2`

<details>
<summary>解答と解説</summary>

**解答:** `x = 5, y = 3`

**解説:**
2式を足すと `2x = 10` → `x = 5`。
代入して `5 + y = 8` → `y = 3`。

> **Programming Link:**
> *   **Elimination:** 変数を消去（Eliminate）して次元を下げる手法です。ガウスの消去法の基礎。
</details>

### Q4. 加減法 (2)
次の連立方程式を解きなさい。
`2x + 3y = 12`
`2x + y = 8`

<details>
<summary>解答と解説</summary>

**解答:** `x = 3, y = 2`

**解説:**
上の式から下の式を引く。
`2y = 4` → `y = 2`。
`2x + 2 = 8` → `2x = 6` → `x = 3`。
</details>

### Q5. 代入法
次の連立方程式を解きなさい。
`y = 2x`
`x + y = 9`

<details>
<summary>解答と解説</summary>

**解答:** `x = 3, y = 6`

**解説:**
下の式の `y` に `2x` を代入。
`x + 2x = 9` → `3x = 9` → `x = 3`。
`y = 2(3) = 6`。

> **Programming Link:**
> *   **Substitution:** 変数展開やマクロ展開と同じ操作です。
</details>

### Q6. 係数を揃える
次の連立方程式を解くために、どちらかの式を何倍かして係数を揃えたい。どうすればよいか。
`3x + 2y = 10` ... (1)
`x - 3y = 5` ... (2)

<details>
<summary>解答と解説</summary>

**解答:** (2)を3倍して `x` の係数を揃える、または (1)を3倍・(2)を2倍して `y` の係数を揃える。

**解説:**
(2) × 3: `3x - 9y = 15`。これと(1)で加減法。
</details>

### Q7. 括弧を含む連立方程式
`2(x - y) = 4`
`3x + 2y = 11`
を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 3, y = 1`

**解説:**
上の式を整理: `2x - 2y = 4` → `x - y = 2`。
これと下の式で解く。
</details>

### Q8. 分数を含む連立方程式
`x/2 + y/3 = 1`
`x - y = 4`
を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 14/5, y = -6/5` ... ではなく整数解になるはず？
再計算:
`3x + 2y = 6`
`x - y = 4` → `y = x - 4`
`3x + 2(x - 4) = 6`
`5x - 8 = 6`
`5x = 14` → `x = 14/5`
計算は合っています。
（問題設定によっては整数にならないこともありますが、基本問題としては珍しいです。確認: `14/10 - 6/15 = 42/30 - 12/30 = 30/30 = 1` OK）
</details>

### Q9. A = B = C の形
方程式 `2x + y = x - y = 3` を連立方程式の形に直しなさい。

<details>
<summary>解答と解説</summary>

**解答:**
`2x + y = 3`
`x - y = 3`

**解説:**
`A = C`, `B = C` の形にするのが一番簡単。
</details>

### Q10. 解がない、無数にある場合
`x + y = 5` と `x + y = 6` の連立方程式の解はどうなりますか。

<details>
<summary>解答と解説</summary>

**解答:** 解なし (No solution)

**解説:**
平行な2直線は交わらない。
逆に `x + y = 5` と `2x + 2y = 10` なら解は無数にある。

> **Programming Link:**
> *   **Unsatisfiable:** 論理式が充足不能（UNSAT）な状態です。矛盾した制約条件。
</details>

## Phase 2: Applied Mastery (応用と実践) - 10問

**「モデル化」**: 現実の問題を連立方程式というモデルに落とし込む訓練です。

### Q11. 個数と代金
1個100円のパンと1個150円のおにぎりを合わせて10個買ったら、代金は1200円でした。それぞれ何個買いましたか。

<details>
<summary>解答と解説</summary>

**解答:** パン6個、おにぎり4個

**Tips:**
パン `x`、おにぎり `y`。
`x + y = 10`
`100x + 150y = 1200`
</details>

### Q12. 2桁の自然数
2桁の自然数があります。各位の数の和は12で、十の位と一の位を入れかえると元の数より18大きくなります。元の数は？

<details>
<summary>解答と解説</summary>

**解答:** 57

**Tips:**
十の位 `x`、一の位 `y`。
`x + y = 12`
`(10y + x) - (10x + y) = 18`
整理すると `9y - 9x = 18` → `y - x = 2`。
`x + y = 12` と連立して `2y = 14` → `y = 7, x = 5`。
</details>

### Q13. 速さ・時間・道のり
A地からB地を通ってC地まで行きます。全体で20km。A〜Bは時速4km、B〜Cは時速5kmで歩いたら、全体で4時間30分かかりました。A〜B、B〜Cの道のりは？

<details>
<summary>解答と解説</summary>

**解答:** A〜B: 10km, B〜C: 10km

**Tips:**
道のりを `x, y` とする。
`x + y = 20`
`x/4 + y/5 = 4.5` (または `9/2`)
下の式を20倍: `5x + 4y = 90`
`4x + 4y = 80`
引き算して `x = 10`。よって `y = 10`。
</details>

### Q14. 食塩水
8%の食塩水と5%の食塩水を混ぜて、6%の食塩水を300g作りたい。それぞれ何g必要ですか。

<details>
<summary>解答と解説</summary>

**解答:** 8%: 100g, 5%: 200g

**Tips:**
`x + y = 300`
`0.08x + 0.05y = 300 × 0.06`
`8x + 5y = 1800`
`5x + 5y = 1500`
`3x = 300` → `x = 100`。
</details>

### Q15. 昨年の生徒数
ある学校の昨年の生徒数は男女合わせて600人。今年は男子が10%増え、女子が5%減り、全体で15人増えました。昨年の男子、女子の人数は？

<details>
<summary>解答と解説</summary>

**解答:** 男子300人、女子300人

**Tips:**
`x + y = 600`
`0.10x - 0.05y = 15`
`10x - 5y = 1500` → `2x - y = 300`
`x + y = 600` と足して `3x = 900` → `x = 300`。
</details>

### Q16. 割合（定価と原価）
ある商品に原価の3割の利益を見込んで定価をつけましたが、売れないので定価の2割引きで売ったら、利益は20円でした。原価を求めなさい。
（※これは一元一次方程式でも解けますが、定価をyとして連立させてもよい）

<details>
<summary>解答と解説</summary>

**解答:** 500円

**Tips:**
原価 `x`。定価 `y`。
`y = 1.3x`
`0.8y - x = 20` （売値 - 原価 = 利益）
代入: `0.8(1.3x) - x = 20`
`1.04x - x = 20`
`0.04x = 20`
`x = 500`
</details>

### Q17. トンネルと列車
ある列車が、長さ300mの鉄橋を渡り始めてから渡り終わるまでに20秒かかりました。また、長さ1000mのトンネルに入り始めてから完全に出るまでに55秒かかりました。列車の長さと速さを求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** 長さ: 100m, 速さ: 秒速20m

**Tips:**
長さ `x` m、速さ `y` m/s。
「渡り終わる」=「鉄橋 + 列車」の距離を進む。
`300 + x = 20y`
`1000 + x = 55y`
下から上を引く: `700 = 35y` → `y = 20`。
`300 + x = 400` → `x = 100`。
</details>

### Q18. 仕事算
ある仕事をAさん1人ですると10日、Bさん1人ですると15日かかります。2人で協力して何日か働いた後、残りをBさん1人で3日かけて仕上げました。2人で働いたのは何日ですか。

<details>
<summary>解答と解説</summary>

**解答:** 4日

**Tips:**
全体の仕事量を `1` とする。
Aの能力 `1/10`、Bの能力 `1/15`。
2人で `x` 日、Bだけで `3` 日。
`(1/10 + 1/15)x + 1/15 × 3 = 1`
`(3/30 + 2/30)x + 1/5 = 1`
`1/6 x = 4/5`
`x = 24/5 = 4.8`日
...問題設定を整数解になるように調整すべきでした。
もし「Bさん1人で5日」なら:
`1/6 x + 1/3 = 1` → `1/6 x = 2/3` → `x = 4`。
**（修正）**: 問題文を「残りをBさん1人で5日」と読み替えてください。そうすれば4日になります。
</details>

### Q19. 年齢の連立方程式
現在、父と子の年齢の和は48歳。5年後、父の年齢は子の年齢の3倍より4歳少なくなります。現在のそれぞれの年齢は？

<details>
<summary>解答と解説</summary>

**解答:** 父38歳、子10歳

**Tips:**
`x + y = 48`
`x + 5 = 3(y + 5) - 4`
</details>

### Q20. 3元連立方程式（発展）
`x + y = 3`, `y + z = 5`, `z + x = 4` を解きなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 1, y = 2, z = 3`

**Tips:**
全て足す: `2(x + y + z) = 12` → `x + y + z = 6`。
ここから各式を引く。
`z = 6 - 3 = 3`
`x = 6 - 5 = 1`
`y = 6 - 4 = 2`

> **Programming Link:**
> *   **Symmetry:** 対称性を利用した解法です。
</details>
