## Phase 1: Foundational Check (基礎の確認) - 10問

**「型システムの拡張」**: 有理数（Rational）から実数（Real）への拡張です。`float` や `double` の世界（近似値）ではなく、代数的数としての厳密な扱いを学びます。

### Q1. 平方根の定義
2乗すると `a` になる数を、`a` の何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 平方根 (Square Root)

**解説:**
`x^2 = a` を満たす `x`。
正の数 `a` には、正と負の2つの平方根がある。

> **Programming Link:**
> *   **Math.Sqrt:** 多くの言語の `sqrt` 関数は正の平方根のみを返しますが、数学的な解は2つあります。
</details>

### Q2. 根号の利用
5の平方根を、根号 `√` を使って表しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `±√5`

**解説:**
`√5` と `-√5` の両方。
`√25` は `5` （正の方だけを指す記号）。
</details>

### Q3. 無理数
有理数（整数 `m` と `0` でない整数 `n` を使って `m/n` と表せる数）ではない実数を何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 無理数 (Irrational Number)

**解説:**
循環しない無限小数になる。
例: `√2`, `π`。

> **Programming Link:**
> *   **Floating Point:** コンピュータでは無理数を有限のビット数で表現するため、必ず誤差（丸め誤差）が生じます。
</details>

### Q4. 平方根の大小
`√3` と `2` の大小を不等号で表しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `√3 < 2`

**解説:**
2乗して比較する。
`(√3)^2 = 3`
`2^2 = 4`
`3 < 4` なので `√3 < 2`。
</details>

### Q5. 根号を外す
`√( (-3)^2 )` の値はいくつですか。

<details>
<summary>解答と解説</summary>

**解答:** 3

**解説:**
`√9 = 3`。
一般に `√(a^2) = |a|`。
`-3` ではないことに注意（`√` の値は常に非負）。
</details>

### Q6. 平方根の積
`√2 × √3` を計算しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `√6`

**解説:**
`√a × √b = √(ab)` （a, b > 0）。
</details>

### Q7. 平方根の商
`√10 ÷ √2` を計算しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `√5`

**解説:**
`√a ÷ √b = √(a/b)`。
</details>

### Q8. 素因数分解と根号
`√12` を `a√b` の形にしなさい。

<details>
<summary>解答と解説</summary>

**解答:** `2√3`

**解説:**
`√12 = √(2^2 × 3) = 2√3`。
「中身をできるだけ小さくする」のが正規化（Normalization）。
</details>

### Q9. 分母の有理化
`1 / √2` の分母を有理化しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `√2 / 2`

**解説:**
分母と分子に `√2` を掛ける。
`√2 / (√2 × √2) = √2 / 2`。

> **Programming Link:**
> *   **Canonical Form:** 数式の標準形です。比較や計算のために形式を統一します。
</details>

### Q10. 平方根の近似値
`√2`、`√3`、`√5` のおよその値は？（語呂合わせ）

<details>
<summary>解答と解説</summary>

**解答:**
`√2 ≒ 1.414` （一夜一夜に人見頃）
`√3 ≒ 1.732` （人並みに奢れや）
`√5 ≒ 2.236` （富士山麓オウム鳴く）
</details>

## Phase 2: Applied Mastery (応用と実践) - 10問

**「演算規則の拡張」**: 文字式と同じルール（分配法則など）が平方根にも適用できます。

### Q11. 平方根の加減
`3√2 + 5√2` を計算しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `8√2`

**Tips:**
`√2` を文字 `x` と同じように扱う（同類項をまとめる）。
`3x + 5x = 8x`。
</details>

### Q12. 異なる平方根の加減
`√12 + √27` を計算しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `5√3`

**Tips:**
まず `a√b` の形に直す。
`2√3 + 3√3 = 5√3`。
**「困ったときは『定義（素因数分解）』に還れ」**。
</details>

### Q13. 分配法則
`√2 (√3 + √2)` を計算しなさい。

<details>
<summary>解答と解説</summary>

**解答:** `√6 + 2`

**Tips:**
`√2 × √3 + √2 × √2 = √6 + 2`。
</details>

### Q14. 乗法公式の利用
`(√5 + 2)(√5 - 2)` を計算しなさい。

<details>
<summary>解答と解説</summary>

**解答:** 1

**Tips:**
`(a + b)(a - b) = a^2 - b^2` を利用。
`5 - 4 = 1`。
分母の有理化（`1 / (√5 + 2)` など）でよく使うテクニックです。
</details>

### Q15. 式の値（対称式）
`x = √3 + √2`, `y = √3 - √2` のとき、`x^2 - y^2` の値を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `4√6`

**Tips:**
因数分解してから代入。
`(x + y)(x - y)`
`x + y = 2√3`
`x - y = 2√2`
`2√3 × 2√2 = 4√6`。
</details>

### Q16. 整数部分と小数部分
`√7` の整数部分を `a`、小数部分を `b` とするとき、`a` と `b` の値を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `a = 2`, `b = √7 - 2`

**Tips:**
`√4 < √7 < √9` より `2 < √7 < 3`。
よって整数部分は2。
小数部分は全体から整数部分を引いたもの。
</details>

### Q17. 平方根の大小（応用）
`√n < 3` を満たす自然数 `n` は何個ありますか。

<details>
<summary>解答と解説</summary>

**解答:** 8個

**Tips:**
2乗して考える。
`n < 9`。
自然数なので `1, 2, ..., 8` の8個。
</details>

### Q18. 有理数になる条件
`√(24n)` が自然数になるような、最も小さい自然数 `n` を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `n = 6`

**Tips:**
素因数分解: `24 = 2^3 × 3`。
指数をすべて偶数にすればルートが外れる。
足りないのは `2` と `3`。
`n = 2 × 3 = 6`。
`√(2^4 × 3^2) = √(144) = 12`。

> **Programming Link:**
> *   **Parity:** パリティチェック（偶奇性）の考え方です。
</details>

### Q19. 体積と平方根
体積が `360` cm^3 の正四角柱があります。高さが `10` cmのとき、底面の正方形の1辺の長さは何cmですか。

<details>
<summary>解答と解説</summary>

**解答:** `6` cm

**Tips:**
底面積 `S = 360 / 10 = 36`。
一辺 `x` とすると `x^2 = 36`。
`x = 6` （長さなので正）。
もし体積が `300` なら `x = √30` cm。
</details>

### Q20. 黄金比（予習）
正五角形の対角線と辺の比などに現れる `(1 + √5) / 2` という値を何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 黄金比 (Golden Ratio)

**Tips:**
近似値は `1.618`。
フィボナッチ数列の隣り合う項の比の極限値でもある。
</details>
