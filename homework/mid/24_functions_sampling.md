## Phase 1: Foundational Check (基礎の確認) - 10問

**「非線形性 (Non-linearity)」**: 入力が2倍になっても出力は2倍にならない（4倍になる）世界です。現実の物理現象（加速、面積、エネルギー）の多くは非線形です。

### Q1. 二次関数の定義
`y` が `x` の関数で、`y = ax^2` （`a`は定数）と表されるとき、`y` は `x` の何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 二乗に比例する

**解説:**
`y` は `x` の二次関数である、ともいう。
</details>

### Q2. グラフの形状
`y = ax^2` のグラフはどのような曲線になりますか。

<details>
<summary>解答と解説</summary>

**解答:** 放物線 (Parabola)

**解説:**
物を投げたときの軌跡。
軸に対して対称（線対称）。

> **Programming Link:**
> *   **Quadratic Curve:** ベジェ曲線（Quadratic Bezier）などで使われる基本的な曲線です。
</details>

### Q3. 係数とグラフの向き
`a > 0` のとき、グラフはどちらに開いていますか。

<details>
<summary>解答と解説</summary>

**解答:** 上に開いている（下に凸）

**解説:**
谷の形。最小値を持つ。
</details>

### Q4. 係数とグラフの開き具合
`a` の絶対値が大きいほど、グラフの開き具合はどうなりますか。

<details>
<summary>解答と解説</summary>

**解答:** 狭くなる（急になる）

**解説:**
変化が激しくなる。
</details>

### Q5. 変化の割合
一次関数と異なり、二次関数 `y = ax^2` の変化の割合は一定ではありません。`x` が `p` から `q` まで増加するときの変化の割合を求める公式（裏技）は？

<details>
<summary>解答と解説</summary>

**解答:** `a(p + q)`

**解説:**
`a(q^2 - p^2) / (q - p) = a(q + p)(q - p) / (q - p) = a(p + q)`。
計算が速くなるので覚えておくと吉。
</details>

### Q6. 変域と最大・最小
`y = x^2` について、`-2 ≦ x ≦ 1` のときの `y` の変域は？

<details>
<summary>解答と解説</summary>

**解答:** `0 ≦ y ≦ 4`

**解説:**
`x = -2` のとき `y = 4`（最大）。
`x = 0` を含むので、最小値は `0`（`x = 1` のときの `1` ではない！）。
頂点を含むかどうかのチェック（境界値テスト）が重要。
</details>

### Q7. 全数調査と標本調査
調査対象のすべてを調べることを全数調査、一部を取り出して調べることを何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 標本調査 (Sampling)

**解説:**
コストや時間の制約、または破壊検査（電球の寿命など）の場合に用いられる。
</details>

### Q8. 母集団と標本
調査対象全体を母集団 (Population)、取り出した一部を何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 標本 (Sample)

**解説:**
標本のサイズ（大きさ）を `n` とする。
</details>

### Q9. 無作為抽出
標本を偏りなく選ぶ方法を何といいますか。

<details>
<summary>解答と解説</summary>

**解答:** 無作為抽出 (Random Sampling)

**解説:**
乱数などを用いる。
偏り（Bias）があると正しい推測ができない。
</details>

### Q10. 標本平均と母平均
標本から求めた平均値（標本平均）は、母集団の平均値（母平均）とどのような関係にありますか。

<details>
<summary>解答と解説</summary>

**解答:** 近似する（標本サイズが大きいほど近づく）

**解説:**
大数の法則。
</details>

## Phase 2: Applied Mastery (応用と実践) - 10問

**「シミュレーション」**: 関数モデルを使って物理現象を予測したり、サンプリングによって全体像を推測したりします。

### Q11. 制動距離
自動車の制動距離（ブレーキがきき始めてから止まるまでの距離）は、速さの2乗に比例します。速さが2倍になると、制動距離は何倍になりますか。

<details>
<summary>解答と解説</summary>

**解答:** 4倍

**Tips:**
`y = ax^2`。
`x` が2倍なら `y` は `2^2 = 4` 倍。
事故のリスクが非線形に増大することを意味する。
</details>

### Q12. 自由落下
物体が落下する距離 `y` mは、時間 `x` 秒の2乗に比例します（`y = 4.9x^2`）。3秒後までに落下する距離は？

<details>
<summary>解答と解説</summary>

**解答:** `44.1m`

**Tips:**
`y = 4.9 × 3^2 = 4.9 × 9 = 44.1`。
</details>

### Q13. 平均の速さ
上記の自由落下において、1秒後から3秒後までの平均の速さは？

<details>
<summary>解答と解説</summary>

**解答:** `19.6 m/s`

**Tips:**
変化の割合 `a(p + q)` を使う。
`4.9(1 + 3) = 4.9 × 4 = 19.6`。
</details>

### Q14. 放物線と直線の交点
`y = x^2` と `y = x + 2` の交点の座標を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `(-1, 1), (2, 4)`

**Tips:**
`x^2 = x + 2`
`x^2 - x - 2 = 0`
`(x - 2)(x + 1) = 0`
`x = 2, -1`。
</details>

### Q15. 変域からの係数決定
`y = ax^2` で、`x` の変域が `-2 ≦ x ≦ 3` のとき、`y` の変域が `-18 ≦ y ≦ 0` でした。`a` の値は？

<details>
<summary>解答と解説</summary>

**解答:** `a = -2`

**Tips:**
最大値が0なので、上に凸（`a < 0`）。
`x` の絶対値が大きい方（`3`）で最小値 `-18` をとる。
`-18 = a × 3^2`
`9a = -18`
`a = -2`。
</details>

### Q16. 捕獲再捕獲法
池の魚の数を調べるため、50匹捕まえて印をつけて戻しました。数日後、30匹捕まえたら印のついた魚が3匹いました。池全体の魚はおよそ何匹と推定されますか。

<details>
<summary>解答と解説</summary>

**解答:** およそ500匹

**Tips:**
全体を `N` とする。
`3 : 30 = 50 : N` （標本内の割合 = 母集団内の割合）
`1 : 10 = 50 : N`
`N = 500`。
</details>

### Q17. 乱数シミュレーション
袋の中に赤玉と白玉が入っています。中身を見ずに100回取り出しては戻す実験をしたところ、赤玉が30回出ました。袋の中の赤玉の割合は？

<details>
<summary>解答と解説</summary>

**解答:** およそ30% (0.3)

**Tips:**
相対度数は確率に収束する。
</details>

### Q18. グラフの変形（平行移動）
`y = x^2` のグラフをx軸方向に2、y軸方向に3だけ平行移動したグラフの式は？（高校範囲だが概念として）

<details>
<summary>解答と解説</summary>

**解答:** `y = (x - 2)^2 + 3`

**Tips:**
頂点が `(0, 0)` から `(2, 3)` に移動する。
</details>

### Q19. 放物線の相似
すべての放物線 `y = ax^2` は互いに相似ですか。

<details>
<summary>解答と解説</summary>

**解答:** 相似である

**Tips:**
拡大縮小すれば重なる。
円がすべて相似であるのと同じ。
</details>
