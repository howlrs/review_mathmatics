## Phase 1: Foundational Check (基礎の確認) - 10問

**「『なぜ？』は理解のエンジン」**: 方程式を解く手順（移項など）が、どのような原理（等式の性質）に基づいているかを常に意識してください。

### Q1. 方程式の解の意味
方程式 `2x - 3 = 5` の解として正しいものを、次の中から選びなさい。
`x = 2, x = 3, x = 4`

<details>
<summary>解答と解説</summary>

**解答:** `x = 4`

**解説:**
代入して等式が成り立つか確認する。
`2 × 4 - 3 = 8 - 3 = 5` (成立)

> **Programming Link:**
> *   **Test Case:** 解が正しいかどうかの検証（Verification）は、テストケースを通すことに似ています。`assert_eq!(2*4 - 3, 5)`
</details>

### Q2. 等式の性質
「等式の両辺に同じ数を足しても、等式は成り立つ」。これを数式で表しなさい（A=Bならば...）。

<details>
<summary>解答と解説</summary>

**解答:** `A + C = B + C`

**解説:**
等式の性質は4つ（加法、減法、乗法、除法）。

> **Programming Link:**
> *   **Invariant:** 操作を行っても保たれる性質（不変条件）。方程式を解くプロセスは、解（Truth）を不変に保ちながら式を変形するプロセスです。
</details>

### Q6. 移項を含む方程式
次の方程式を解きなさい。
`3x = 5x + 8`

<details>
<summary>解答と解説</summary>

**解答:** `x = -4`

**解説:**
xを含む項を左辺へ、定数項を右辺へ移項。
`3x - 5x = 8`
`-2x = 8`
`x = -4`
</details>

### Q7. 括弧を含む方程式
次の方程式を解きなさい。
`4(x - 2) = x + 1`

<details>
<summary>解答と解説</summary>

**解答:** `x = 3`

**解説:**
まず括弧を開く（展開）。
`4x - 8 = x + 1`
`3x = 9`
`x = 3`
</details>

### Q8. 小数を含む方程式
次の方程式を解きなさい。
`0.3x + 0.2 = 1.4`

<details>
<summary>解答と解説</summary>

**解答:** `x = 4`

**解説:**
両辺を10倍して整数にする。
`3x + 2 = 14`
`3x = 12`
`x = 4`

> **Programming Link:**
> *   **Normalization:** 扱いやすい形式（整数）にデータを正規化してから処理する手法です。浮動小数点の誤差を避けるためにも有効です。
</details>

### Q9. 分数を含む方程式
次の方程式を解きなさい。
`x/2 - 1/3 = x/6`

<details>
<summary>解答と解説</summary>

**解答:** `x = 1`

**解説:**
両辺に分母の最小公倍数（6）を掛けて分母を払う。
`3x - 2 = x`
`2x = 2`
`x = 1`
</details>

### Q10. 比の方程式
`x : 4 = 3 : 2` のとき、xの値を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** `x = 6`

**解説:**
「外項の積 ＝ 内項の積」
`2x = 12`
`x = 6`

> **Programming Link:**
> *   **Cross Multiplication:** 比率計算やスケーリングで頻出するロジックです。
</details>

## Phase 2: Applied Mastery (応用と実践) - 10問

**「条件を使い切ったか確認せよ」**: 文章題では、与えられた数値や条件をすべて式に反映させる必要があります。

### Q11. 代金の問題
1個120円のりんごと1個80円のみかんを合わせて15個買ったら、代金の合計が1400円でした。りんごは何個買いましたか？

<details>
<summary>解答と解説</summary>

**解答:** 5個

**Tips:**
りんごの個数を `x` とおく。みかんの個数は `15 - x` となる。
`120x + 80(15 - x) = 1400`
これを解くと `40x = 200` → `x = 5`。

> **Programming Link:**
> *   **Variable Scope:** 変数 `x` を定義したら、他の量（みかんの個数）も `x` を使って表現（依存関係を定義）します。
</details>

### Q12. 過不足の問題
生徒に画用紙を配るのに、1人に5枚ずつ配ると10枚余り、6枚ずつ配ると8枚足りません。生徒の人数は何人ですか？

<details>
<summary>解答と解説</summary>

**解答:** 18人

**Tips:**
生徒数を `x` 人とおく。
画用紙の枚数を2通りの式で表して等号で結ぶ。
`5x + 10 = 6x - 8`
`18 = x`

> **Programming Link:**
> *   **Equivalence Checking:** 同じオブジェクト（画用紙の総数）に対して、異なるアクセスパス（配り方）からアプローチし、整合性をチェックしています。
</details>

### Q13. 速さの問題（追いつく）
弟が家を出てから10分後に、兄が自転車で弟を追いかけました。弟の速さは分速60m、兄の速さは分速210mです。兄は出発してから何分後に弟に追いつきますか？

<details>
<summary>解答と解説</summary>

**解答:** 4分後

**Tips:**
兄が走った時間を `x` 分とおく。弟が歩いた時間は `x + 10` 分。
「進んだ道のりが等しい」という等式を作る。
`210x = 60(x + 10)`
`210x = 60x + 600`
`150x = 600`
`x = 4`

> **Programming Link:**
> *   **Simulation:** 時間 `t` をパラメータとして、二つのオブジェクトの位置 `Pos(t)` が一致する時点を求めています。
</details>

### Q14. 割合の問題（増減）
ある学校の昨年の生徒数は男女合わせて500人でした。今年は男子が10%増え、女子が20%減ったため、全体で5人減りました。今年の男子の人数を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** 275人

**Tips:**
昨年の男子を `x` 人とおく。昨年の女子は `500 - x` 人。
増減の人数だけに注目して式を作ると計算が楽です。
`0.1x - 0.2(500 - x) = -5`
`0.1x - 100 + 0.2x = -5`
`0.3x = 95`
`x = 950 / 3` ... 割り切れない？
計算確認:
`0.1x - 100 + 0.2x = -5`
`0.3x = 95`
`x = 316.6...`
問題設定ミスまたは計算ミスを確認。
「全体で5人減った」→ `-5`。
式は正しい。
あ、問題文の数値設定が整数解にならない可能性があります。
再計算:
もし昨年の男子250, 女子250なら、+25, -50 で -25人。
もし昨年の男子300, 女子200なら、+30, -40 で -10人。
もし昨年の男子350, 女子150なら、+35, -30 で +5人。
300と350の間。
`0.3x = 95` → `x = 316.66...`
**（修正）**: 問題設定を「全体で10人減った」に変えると、`0.3x = 90` で `x = 300` となり綺麗です。
ここでは、**「昨年の男子」を求めるのではなく「今年の男子」を求める**点に注意。
もし `x=300` (昨年) なら、今年は `300 * 1.1 = 330` 人。
※この問題は数値設定が微妙でしたが、解き方のロジックは上記の通りです。
（実習としては、`x`が整数にならない時点で「定義（人数は自然数）」に反するため、入力エラーを疑います）
</details>

### Q15. 年齢の問題
現在、父は40歳、子は12歳です。父の年齢が子の年齢の3倍になるのは何年後ですか？

<details>
<summary>解答と解説</summary>

**解答:** 2年後

**Tips:**
x年後とする。
`40 + x = 3(12 + x)`
`40 + x = 36 + 3x`
`4 = 2x`
`x = 2`
</details>

### Q16. 図形と方程式
縦の長さが横の長さより3cm短い長方形があります。周囲の長さが34cmのとき、横の長さを求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** 10cm

**Tips:**
横を `x` cmとする。縦は `x - 3` cm。
周囲 = `(縦 + 横) × 2`
`2(x + x - 3) = 34`
`4x - 6 = 34`
`4x = 40`
`x = 10`
</details>

### Q17. 食塩水の問題
10%の食塩水200gに、5%の食塩水を混ぜて8%の食塩水を作りたい。5%の食塩水を何g混ぜればよいですか？

<details>
<summary>解答と解説</summary>

**解答:** 133.3...g (400/3 g)

**Tips:**
混ぜる量を `x` gとする。
「溶けている食塩の量」で等式を作る（質量保存則）。
`200 × 0.10 + x × 0.05 = (200 + x) × 0.08`
`20 + 0.05x = 16 + 0.08x`
`4 = 0.03x`
`x = 400/3`
（これも割り切れませんが、方程式の練習としては正解です）

> **Programming Link:**
> *   **Weighted Average:** 加重平均の計算です。
</details>

### Q18. 比例式と文章題
ある本を、読んだページ数と残りのページ数の比が 3:2 になるまで読みました。読んだページ数が60ページのとき、この本は全部で何ページですか？

<details>
<summary>解答と解説</summary>

**解答:** 100ページ

**Tips:**
全ページを `x` とするか、残りを `y` としてから足すか。
読んだ : 残り = 3 : 2
読んだ : 全体 = 3 : (3+2) = 3 : 5
`60 : x = 3 : 5`
`3x = 300`
`x = 100`
</details>

### Q19. 規則性と方程式
白い碁石を正方形の形にぎっしり並べたら、全部で `x` 個使いました。一番外側のひとまわりの碁石の個数を `x` を使って表しなさい（一辺の個数 `n` を使って考えてから `x` に変換）。
※これは少し難問です。`x` は平方数（n^2）です。

**Tips:**
一辺の個数を `n` とすると、全体は `n^2`。つまり `n = √x`。
外周は `4n - 4` （角の重複を除く）。
よって `4√x - 4`。
中学1年生では `√` を習っていないため、通常は「一辺をn個として、外周が20個のときの総数」などを問います。
**問題変更:** 一辺に `n` 個並べた正方形の外周の個数が 36個のとき、碁石は全部で何個ありますか？

<details>
<summary>解答と解説</summary>

**解答:** 100個

**解説:**
外周の個数は `4n - 4`。
`4n - 4 = 36`
`4n = 40`
`n = 10`
総数は `10 × 10 = 100`個。
</details>

### Q20. 抽象的な思考
方程式 `ax + b = c` の解が `x = 1` であるとき、`a, b, c` の関係を表す式を作りなさい。

<details>
<summary>解答と解説</summary>

**解答:** `a + b = c`

**Tips:**
解が `x = 1` ということは、`x` に `1` を代入すれば等式が成り立つということ。
`a(1) + b = c`
`a + b = c`

> **Programming Link:**
> *   **Parameter Tuning:** アルゴリズムが特定の出力（x=1）を出すための、パラメータ（a, b, c）の制約条件を求めています。
</details>
