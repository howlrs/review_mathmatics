## Phase 1: Foundational Check (基礎の確認) - 10問

**「距離関数 (Distance Function)」**: ユークリッド空間における距離（ノルム）を定義する、幾何学で最も重要な定理です。

### Q1. 三平方の定理
直角三角形の直角をはさむ2辺の長さを `a, b`、斜辺の長さを `c` とするとき、成り立つ関係式は？

<details>
<summary>解答と解説</summary>

**解答:** `a^2 + b^2 = c^2`

**解説:**
ピタゴラスの定理。
`c = √(a^2 + b^2)` （ユークリッド距離）。

> **Programming Link:**
> *   **Hypot:** 多くの言語に `hypot(x, y)` 関数があり、`sqrt(x*x + y*y)` をオーバーフローなしで計算してくれます。
</details>

### Q2. 斜辺の長さ
2辺が `3cm, 4cm` の直角三角形の斜辺の長さは？

<details>
<summary>解答と解説</summary>

**解答:** `5cm`

**解説:**
`3^2 + 4^2 = 9 + 16 = 25`。
`√25 = 5`。
</details>

### Q3. 他の一辺の長さ
斜辺が `13cm`、他の一辺が `5cm` の直角三角形の残りの辺の長さは？

<details>
<summary>解答と解説</summary>

**解答:** `12cm`

**解説:**
`x^2 + 5^2 = 13^2`
`x^2 + 25 = 169`
`x^2 = 144`
`x = 12`。
</details>

### Q4. ピタゴラス数
3辺がすべて整数になる直角三角形の辺の比（ピタゴラス数）を2組挙げなさい。

<details>
<summary>解答と解説</summary>

**解答:** `(3, 4, 5)`, `(5, 12, 13)`

**解説:**
他には `(8, 15, 17)`, `(7, 24, 25)` など。
覚えておくと計算が速くなる（キャッシュ）。
</details>

### Q5. 直角三角形の判別
3辺が `2, 3, 4` の三角形は直角三角形ですか。

<details>
<summary>解答と解説</summary>

**解答:** 直角三角形ではない

**解説:**
`2^2 + 3^2 = 4 + 9 = 13`。
`4^2 = 16`。
等しくないので直角ではない（鈍角三角形）。
</details>

### Q6. 特別な直角三角形 (1)
三角定規（45度-45度-90度）の3辺の比は？

<details>
<summary>解答と解説</summary>

**解答:** `1 : 1 : √2`

**解説:**
正方形を対角線で切った形。
</details>

### Q7. 特別な直角三角形 (2)
三角定規（30度-60度-90度）の3辺の比は？

<details>
<summary>解答と解説</summary>

**解答:** `1 : 2 : √3` （または `1 : √3 : 2`）

**解説:**
一番短い辺が `1`、斜辺が `2`、残りが `√3`。
正三角形を半分にした形。
</details>

### Q8. 正三角形の高さ
一辺の長さが `a` の正三角形の高さは？

<details>
<summary>解答と解説</summary>

**解答:** `(√3 / 2) a`

**解説:**
`1 : 2 : √3` を使う。
底辺の半分 `a/2` と斜辺 `a` から求める。
</details>

### Q9. 座標平面上の距離
2点 `A(x1, y1)`, `B(x2, y2)` 間の距離を求める公式は？

<details>
<summary>解答と解説</summary>

**解答:** `√((x2 - x1)^2 + (y2 - y1)^2)`

**解説:**
三平方の定理そのもの。
</details>

### Q10. 立体対角線
縦 `a`、横 `b`、高さ `c` の直方体の対角線の長さは？

<details>
<summary>解答と解説</summary>

**解答:** `√(a^2 + b^2 + c^2)`

**解説:**
平面での三平方を2回適用して導出。
3次元ベクトルの大きさ（ノルム）。
</details>

## Phase 2: Applied Mastery (応用と実践) - 10問

**「最短経路探索」**: 空間内での距離計算や、展開図を用いた最短経路問題に応用します。

### Q11. 円錐の高さと体積
母線の長さが `13cm`、底面の半径が `5cm` の円錐の高さと体積を求めなさい。

<details>
<summary>解答と解説</summary>

**解答:** 高さ `12cm`、体積 `100π cm^3`

**Tips:**
高さ `h = √(13^2 - 5^2) = 12`。
体積 `V = 1/3 × π(5^2) × 12 = 100π`。
</details>

### Q12. 最短経路（展開図）
円柱の側面上を、底面の点Aから真上の点Bの反対側まで紐をかけるとき、紐の長さが最短になるのはどのような線ですか。

<details>
<summary>解答と解説</summary>

**解答:** 展開図における直線

**Tips:**
側面（長方形）の対角線になる。
螺旋（Helix）は展開すると直線になる。
</details>

### Q13. 正四面体の高さ
一辺の長さが `6cm` の正四面体の高さは？

<details>
<summary>解答と解説</summary>

**解答:** `2√6 cm`

**Tips:**
頂点から底面に下ろした垂線は、底面の正三角形の重心を通る。
重心までの距離は、中線の `2/3`。
中線 `3√3`。重心まで `2√3`。
高さ `h = √(6^2 - (2√3)^2) = √(36 - 12) = √24 = 2√6`。
</details>

### Q14. 弦の長さ
半径 `5cm` の円の中心から `3cm` の距離にある弦の長さは？

<details>
<summary>解答と解説</summary>

**解答:** `8cm`

**Tips:**
直角三角形を作る。
斜辺 `5`、他の一辺 `3`。
残りの辺は `4`。
弦の長さは `2倍` なので `8`。
</details>

### Q15. 接線の長さ
半径 `3cm` の円外の点P（中心からの距離 `5cm`）から引いた接線の長さは？

<details>
<summary>解答と解説</summary>

**解答:** `4cm`

**Tips:**
接線は半径と垂直に交わる。
斜辺 `5`、半径 `3` の直角三角形。
`√(5^2 - 3^2) = 4`。
</details>

### Q16. 2点間の距離
`A(1, 2)`, `B(4, 6)` の間の距離は？

<details>
<summary>解答と解説</summary>

**解答:** `5`

**Tips:**
`x`の差 `3`, `y`の差 `4`。
`3:4:5` の三角形。
</details>

### Q17. 折り紙の面積
正方形ABCD（一辺10cm）を、頂点Aが辺BCの中点Mに来るように折ったとき、折り目の線分EFの長さは？
（難問）

<details>
<summary>解答と解説</summary>

**解答:** `(未計算、方針のみ)`

**Tips:**
折り目の性質（垂直二等分線）を使う。
AMの垂直二等分線がEF。
相似な直角三角形を見つけて比で解く。
</details>

### Q18. 最短経路（直方体）
縦3、横4、高さ5の直方体の頂点から対角の頂点まで、表面を伝って行く最短距離は？

<details>
<summary>解答と解説</summary>

**解答:** `√74`

**Tips:**
展開図のパターンは3通りある。
1. `√((3+4)^2 + 5^2) = √(49+25) = √74`
2. `√((3+5)^2 + 4^2) = √(64+16) = √80`
3. `√((4+5)^2 + 3^2) = √(81+9) = √90`
最小は `√74`。
</details>

### Q19. 球の切断面
半径 `10cm` の球を、中心から `6cm` の平面で切ったとき、切り口の円の面積は？

<details>
<summary>解答と解説</summary>

**解答:** `64π cm^2`

**Tips:**
切り口の半径 `r = √(10^2 - 6^2) = 8`。
面積 `64π`。
</details>
